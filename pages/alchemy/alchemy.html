Purpose

This project's aim was to improve customers experience in My Account area but also community and Help section. Alchemy was designed to enable users to find relevant information quickly and efficiently. It was created to increase customer engagement, offer values and relevant offer sale prices, grow TalkTalk community but also reduce calls.


Brainstorming and prototyping

Given the creative freedom we were able to create new ways the customers interact with TalkTalk. Our main goal was to integrate TalkTalk's main web applications (My Account, Help, Community) into one experience. A place where customers can find all the help TalkTalk provides. At this stage of the process we should start wire-framing our ideas and creating basic user journeys. We have set up a focus group where we could test and prove our ideas.
